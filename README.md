# vue_fast

> 迅捷点餐,简单的实现了注册、登陆、获取用户信息、获取餐厅列表、以及无限加载列表功能、实现了登陆页面拦截（为方便测试，有效期设置为 30s 进行监测，普通生产情况下可以设置为更长的时间）

## Build Setup

``` bash
# 下载依赖
npm install

# 启动前端服务
npm run dev

# 前端编译
npm run build

# 启动后端服务
npm run server
